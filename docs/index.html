CSPCL

C Rust Lead contributor

CubeSat Space Protocol Convergence Layer — a lightweight library that bridges the Bundle Protocol 7 (BPv7) and the CubeSat Space Protocol (CSP), enabling DTN (Delay/Disruption Tolerant Networking) over space radio links. The project is developed within the DTN-MTP organization (University Space Center of Montpellier).

Project purpose

Space communications suffer from long delays and frequent outages. The DTN/BPv7 protocol addresses this with store-and-forward behavior but requires convergence layers adapted to each physical link. CSPCL fills this gap by allowing BPv7 bundles to be carried over CAN buses (typical for CubeSats) or ZeroMQ (for ground testing), with automatic fragmentation via SFP. The library integrates with two BPv7 implementations: µD3TN (C) and Hardy (Rust).
